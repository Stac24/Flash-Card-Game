Backend for flash card app

Quickstart:
```
git clone https://github.com/Stac24/Flash-Card-Game
cd server
npm install
npm run seed:db
npm run server
```

And then go to localhost:4001/getcards for some sample flashcards

.env.dev is an example for .env
